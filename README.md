**База даних ментального здоров'я** — це персональна інструментальна система для відстеження, структурування та аналізу щоденної активності та ментального стану користувача, яка керується агентом штучного інтелекту.

 Проєкт допомагає фіксувати щоденні події за різними категоріями життя (такими як Сім'я, Робота, Навчання, Здоров'я тощо), оцінювати їхній вплив на загальне самопочуття та генерувати аналітичні звіти для глибшої рефлексії.

> [!NOTE]
> Проєкт перебуває в процесі активної розробки.
