## Типові категорії активностей

| Категорія | Опис / Приклади активностей |
| :--- | :--- |
| **Сім'я** | Спільне дозвілля, спілкування, взаємна підтримка, сімейні конфлікти тощо |
| **Робота** | Досягнення, робочі завдання, наради, стрес, перевантаження тощо |
| **Навчання** | Отримання нових знань, курси, вебінари, читання професійної літератури тощо |
| **Хобі** | Творчість, улюблена справа, відпочинок, захоплення тощо |
| **Саморозвиток** | Медитація, рефлексія, психотерапія, особистісні практики тощо |
| **Здоров'я** | Сон, харчування, самопочуття, лікування, профілактика тощо |
| **Спорт** | Фізичні тренування, розминка, йога, біг, активні заняття тощо |
| **Побут** | Прибирання, покупки, приготування їжі, вирішення домашніх справ тощо |
| **Адвокатський іспит** | Підготовка до тестування, вирішення юридичних кейсів, стрес перед іспитом, складання тестування |
| **Відновлення** | Повноцінний відпочинок, інформаційний детокс, час на самоті, відновлення сил тощо |
